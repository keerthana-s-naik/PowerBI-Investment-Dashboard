# PowerBI-Investment-Dashboard
Power BI dashboard analyzing investment dataset

#Overview
This project analyzes investment dataset using Power BI and presents insights using interactive visualisations.

There are 7 Different analyses made in this project, like.,
> Exploration of the dataset
> Gender based analysis
> Objective analysis
> Investment duration and frequency
> Source of information
> Reasons for information
> And a dashboard with an overall combined analysis


#Features of the Dashboard
> Interactive dashboard
> Data filtering and slicers
> Visual representation of key metrics
> Insightful charts and graphs
